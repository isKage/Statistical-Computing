## Project Reproduction

[中文](./README_zh.md) / [doc](./pdf/hw4.pdf)

First, configure the environment.
Then, navigate to the project root directory.
Finally, enter the command in the terminal to fully reproduce all results of this project.

- macOS / Linux: Enter in the `Terminal`  
```BASH
bash main.sh
```
- Windows: Enter in `cmd`  
```BASH
.\main.bat
```
Or enter in `PowerShell`  
```BASH
.\main.ps1
```
Alternatively, simply double-click the `main.bat` script.
